# Proyecto: Landing Page de Viajes

## Descripción General

Este repositorio contiene una **Landing Page básica** desarrollada como proyecto final del curso de **Introducción al Desarrollo Web Front-End: HTML y CSS desde cero por Crehana**.

El objetivo principal fue aplicar los conceptos fundamentales de estructuración web y estilos para construir una página de una sola vista (viajes/turismo). Es mi primer proyecto para documentar mi proceso de aprendizaje en el desarrollo web.

## Tecnologías Utilizadas

* **HTML5:** Usado para la estructura semántica del contenido (`header`, `main`, `section`, `footer`).
* **CSS3:** Empleado para el diseño, estilizado y maquetación de la página.

## Características Clave del Proyecto

* **Estructura Semántica Sencilla:** Implementación básica de las etiquetas de HTML5 para organizar el contenido (Encabezado, Banner, Secciones de Contenido y Pie de página).
* **Diseño de Layout:** Uso de **Flexbox** en la navegación (`.menu`) y el pie de página (`.center-footer .container`) para alinear y distribuir elementos.
* **Estilos:** Aplicación de tipografía personalizada (`Open Sans` de Google Fonts), manejo de colores y creación de un banner con imagen de fondo (`background-image`).
* **Diseño Básico Adaptativo:** Se incluye una `@media query` para mejorar la visualización en pantallas de escritorio grandes (a partir de `769px`), ajustando las secciones "Nosotros".
